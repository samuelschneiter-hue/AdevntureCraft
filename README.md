# Item Codex

A read-only in-game item preview catalog built with TanStack Start and deployed on Netlify.

Visitors can browse all available in-game items, filter by category and rarity, and view detailed item information including stats and listed prices — with no purchasing functionality.

## Key Technologies

- **TanStack Start** — full-stack React SSR framework
- **TanStack Router** — file-based routing with type-safe params
- **Tailwind CSS v4** — utility styling
- **Lucide React** — item type icons
- **Netlify** — hosting and deployment

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts on [http://localhost:3000](http://localhost:3000).

Or use the Netlify CLI for full platform emulation:

```bash
netlify dev
```
